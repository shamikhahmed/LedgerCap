# StundsOS

Personal Wealth Operating System — PSX stocks, Meezan funds, portfolio tracking.

**Live:** https://shamikhahmed.github.io/stundsOS/

## Features

- Portfolio dashboard with net worth and P&L
- PSX + Islamic fund holdings with live Yahoo prices
- Transaction ledger (buy/sell/contribution/salary)
- Income projections and financial freedom target
- Insights: concentration risk, contribution streaks

## Install on iPhone

Safari → Share → **Add to Home Screen**

## Local dev

```bash
python3 -m http.server 8765
```
